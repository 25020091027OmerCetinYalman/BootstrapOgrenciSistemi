# 🎓 Öğrenci Yönetim Sistemi Paneli (Bootstrap & SB Admin 2)

Bu proje, Yazılım Mühendisliği müfredatı kapsamında, Bootstrap framework'ü ve SB Admin 2 şablonu özelleştirilerek geliştirilmiş akademik bir arayüz projesidir.

### 🚀 Özellikler:
- **Responsive Tasarım:** Tüm cihazlarla (Mobil, Tablet, Masaüstü) tam uyumlu Bootstrap Grid yapısı.
- **Özelleştirilmiş Giriş:** Üniversite logosu ile entegre edilmiş, şık ve interaktif Giriş/Şifre Hatırlatma arayüzü (`login.html`).
- **Dashboard (Genel Özet):** Akademik kadro ve öğrenci temsilcisi bilgilerini içeren, görsel kartlara dayalı modern panel (`index.html`).
- **Öğrenci Yönetimi (Tables):** Filtrelenebilir öğrenci listesi, devamsızlık takibi ve akademik durum sütunlarını içeren detaylı veri tablosu (`tables.html`).
- **Kod Temizliği:** Şablondaki gereksiz, İngilizce ve içi boş kısımlar temizlenerek proje tamamen amaca yönelik hale getirilmiştir.

### 🛠️ Kullanılan Teknolojiler:
- HTML5 / CSS3 (Bootstrap v4.x)
- JavaScript (jQuery)
- SB Admin 2 Template
